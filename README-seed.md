# Скрипт генерации тестовых данных

Этот скрипт создает администратора, пользователей с разными ролями, курсы, уроки, события, посты и комментарии для тестирования приложения.

## Запуск скрипта

Для запуска скрипта выполните команду:

```bash
npm run seed
```

или 

```bash
yarn seed
```

## Что создает скрипт

1. **Администратора** с логином `111` и паролем `123`
2. **Пользователей** с разными ролями:
   - 5 пользователей с ролями: student, teacher, moderator (по очереди)
   - Логины: `user1`, `user2`, `user3`, `user4`, `user5`
   - Пароли: `password123` для всех обычных пользователей
3. **Курсы** (3 штуки) с уровнями: beginner, intermediate, advanced
4. **Уроки** (по 3 урока для каждого курса)
5. **События** (4 штуки)
6. **Посты** (5 штук) с разными категориями
7. **Комментарии** (по 2 комментария к каждому посту от разных пользователей)

## Структура данных

- Администратор имеет роль `admin`
- Пользователи распределены по ролям: `student`, `teacher`, `moderator`
- Курсы связаны с уроками через поле `course`
- Посты и комментарии связаны через поле `post`
- Все данные создаются с корректными ссылками между собой

## Важно

- Скрипт удаляет все существующие данные перед созданием новых
- Подключение к базе данных осуществляется через переменную окружения `MONGO_URI`
- Если переменная `MONGO_URI` не установлена, скрипт может не работать корректно
